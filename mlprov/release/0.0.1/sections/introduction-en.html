Introduction back to ToC

We use the W3C PROV standard as a basis [CITE], in particular the PROV-O ontology which we extend for ML models. PROV represents resources (entities), actions (activities), and actors (agents). PROV also represents their relationships: an entity was used by an activity, a new entity was generated by an activity, an agent was associated with an activity, and so on.
